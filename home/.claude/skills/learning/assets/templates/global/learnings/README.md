# Learnings — 使い方

このフォルダは、複数のプロジェクトで使える学びを置く場所。`index.md` は読み込み時に常に参照されるので短く保ち、詳しい説明はこのファイルに書く。

## 置き方

- リポジトリ固有の学びは、そのリポジトリの `LEARNINGS.md` に書く。
- 複数のプロジェクトで使える学びだけ、ここ（`~/.agents/learnings/`）に置く。
- 学びの本体は `<topic>.md` に書き、`index.md` には1行の要約だけ（「タイトル — ファイル名 — 1行要約」）追記する。

## ここに移す条件

- `LEARNINGS.md` の学びが、このリポジトリの外でも使えると判断できる。
- 「どんな場面で使えるか」と「使えない場面」が書ける。

## 学びの書き方

`learning` スキルの形式に従う（タイトル / 学び / 根拠 / 適用範囲 / 例外・前提）。
