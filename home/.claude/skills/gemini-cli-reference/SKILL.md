# Gemini CLI Reference

このスキルは Gemini CLI の公式ドキュメントリファレンスです。
ユーザーからGemini CLIに関する質問（使い方、コマンド、設定、トラブルシューティング、スキルの作成方法など）があった場合、添付の `references/` ディレクトリ内のドキュメントを参照して正確な情報を提供してください。

## 使い方
1. **ドキュメントの検索**: ユーザーの質問内容に関連しそうなドキュメントを `references/` ディレクトリ配下から検索・参照します（例: コマンドなら `cli-reference.md` や `commands.md`、設定なら `settings.md` や `configuration.md` など）。
2. **正確な回答の生成**: ドキュメントの内容に基づき、Gemini CLIの最新の仕様に忠実な回答を作成します。存在しない機能や推測による回答は避けてください。
3. **実例の提示**: 設定例やコマンド例がドキュメントにある場合は、それらを積極的に引用してわかりやすく提示してください。
