# Agents Hooks（共有スクリプト）

フックから実行される、エージェント中立な共有スクリプトの置き場。`~/.agents/hooks/` として全エージェントから参照する。

## 方針（cross-agent 制約）

フックはエージェントが提供する機能で、起動機構も設定形式もエージェントごとに異なる（Claude Code = `settings.json` / Pi = TypeScript Extension / Codex = 別機構）。そのため**ロジックはこのディレクトリの共有スクリプトに寄せ、各エージェントのアダプタは「このスクリプトを呼ぶだけの薄い定義」に限定する**。

- ロジックの実体をここに1つだけ置く（移植性・保守性）。
- 各エージェントのアダプタは最小限にとどめる。
- 起動機構そのものは移植できない前提で設計する（依存の3プリミティブのうちフックは ✕ 評価）。
