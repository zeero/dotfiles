# Agents Hooks（共有スクリプト）

ツール中立な hook 用の共有スクリプト置き場。`~/.agents/hooks/` として全ツールから参照する。

## 方針（cross-tool 制約）

フックの起動機構はツールごとに異なる（Claude Code = `settings.json` / Pi = TypeScript Extension / Codex = 別機構）。そのため**ロジックはこのディレクトリの共有スクリプトに寄せ、各ツール側の hook / extension は「このスクリプトを呼ぶだけの薄いアダプタ」に限定する**。

- ロジックの単一実体をここに置く（移植性・保守性）。
- 各ツールの起動定義は最小限のアダプタにとどめる。
- 起動機構そのものは移植できない前提で設計する（依存の3プリミティブのうちフックは ✕ 評価）。
